# BellezApp

Aplicación web para la gestión de turnos en una peluquería. Permite a los usuarios registrarse, iniciar sesión, reservar turnos y administrar su perfil. Incluye un panel de administración para gestionar servicios y turnos.

## Tecnologías utilizadas

**Frontend**

* React
* Vite
* CSS

**Backend**

* Node.js
* Express

**Base de datos**

* SQLite

## Estructura del proyecto

```
BellezApp
├── BellezApp-Frontend
└── BellezApp-Backend
```

## Instalación y ejecución

### 1. Clonar el repositorio

```
git clone https://github.com/tobiascrocus/BellezApp.git
cd BellezApp
```

### 2. Instalar dependencias

Backend:

```
cd BellezApp-Backend
npm install
```

Frontend:

```
cd ../BellezApp-Frontend
npm install
```

### 3. Ejecutar la aplicación

Backend:

```
cd BellezApp-Backend
node server.js
```

Frontend:

```
cd BellezApp-Frontend
npm run dev
```

## Funcionalidades principales

* Registro e inicio de sesión de usuarios
* Gestión de turnos
* Panel de administración
* Catálogo de servicios
* Perfil de usuario
